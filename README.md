# Fake Job Detection System

A Machine Learning and NLP based web application that predicts whether a job posting is **Real** or **Fake**.

## Live Demo

🔗 https://fakejobcheck.streamlit.app

---

## Features

* Fake vs Real Job Prediction
* NLP Text Preprocessing
* TF-IDF Vectorization
* Professional Streamlit UI
* Multiple Machine Learning Algorithms Comparison
* Random Forest Best Model
* Real-Time Prediction

## Run Locally

Clone the repository:

```bash
git clone https://github.com/your-username/fake-job-detection.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
